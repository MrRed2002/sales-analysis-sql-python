# Sales Analysis - SQL + Python

This project demonstrates a sales analysis system built using:

- Python
- SQLite
- SQL (JOIN, GROUP BY, WHERE, Aggregations)

# Features

- Relational database structure (Products & Sales)
- Foreign key relationship
- JOIN queries
- Revenue calculation
- Sales report by category
- Modular project structure

# Database Structure

@Products
- id
- name
- category

@Sales
- id
- product_id (Foreign Key)
- quantity
- price

# Example Output

- Total revenue per product
- Filtered sales by category
- Aggregated quantity and revenue
